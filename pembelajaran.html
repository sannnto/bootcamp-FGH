Learn HTML

HTML adalah bahasa markup standar untuk halaman web.

Dengan HTML kamu bisa membuat situs web sendiri.

HTML itu mudah dipelajari â€“ kamu akan menikmatinya!

Pembelajaran Mudah dengan "Try it Yourself"

Dengan editor "Try it Yourself", kamu dapat mengedit kode HTML dan melihat hasilnya secara langsung.

Example

<!DOCTYPE html>
<html>
<head>
  <title>Page Title</title>
</head>
<body>

  <h1>This is a Heading</h1>
  <p>This is a paragraph.</p>

</body>
</html>

Learn CSS

CSS digunakan untuk mendesain dan membuat tampilan halaman web lebih menarik.

CSS memungkinkan kamu untuk mengubah warna, tata letak, dan animasi elemen HTML.

Pembelajaran CSS

Contoh sederhana penggunaan CSS untuk mewarnai teks:

Example

<!DOCTYPE html>
<html>
<head>
<style>
h1 {
  color: blue;
}
</style>
</head>
<body>

<h1>This is a blue heading</h1>

</body>
</html>

Learn JavaScript

JavaScript membuat halaman web menjadi interaktif.

Kamu bisa menambahkan logika, reaksi terhadap klik, dan manipulasi data dengan JavaScript.

Pembelajaran JavaScript

Contoh sederhana JavaScript yang menampilkan alert:

Example

<!DOCTYPE html>
<html>
<body>

<h1>My First JavaScript</h1>
<button onclick="alert('Hello World!')">Click Me</button>

</body>
</html>
